# IslandLink Sales Distribution System

## 📌 Project Aim
The main aim of this project is to design and develop a centralised sales distribution management system for IslandLink Sales Distribution Network (ISDN) in order to automate ordering, inventory management, delivery tracking, and payment processes, and to improve overall efficiency and customer service.

---

## 🎯 Project Objectives
1. Design a web-based system that allows customers to view products and place orders online.
2. Implement delivery scheduling and tracking to improve delivery visibility.
3. Apply role-based access control to secure the system for different user types.
4. Test the system to ensure it works correctly and meets user requirements.
5. Prepare system documentation and a user manual for end users.

---

## 🛠 Technologies Used
- PHP
- MySQL
- HTML, CSS, TailwindCSS
- JavaScript
- Google Maps API
- PHPMailer
- FPDF
- XAMPP Server

---

## ⚙️ How to Run the Project

1. Install XAMPP
2. Copy project folder into:
C:\xampp\htdocs\ase\

3. Import database file into phpMyAdmin
4. Start Apache and MySQL in XAMPP
5. Open browser and go to:
http://localhost/ase/

## 👨‍💻 Features
- Customer product browsing and ordering
- Inventory management system
- Delivery scheduling & tracking
- Role-based user system (Admin / Staff / Driver / Customer)
- Invoice generation (PDF)
- Reports generation system
